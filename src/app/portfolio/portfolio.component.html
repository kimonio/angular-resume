Mobile App for Fun Rewards
  • https://funrewards.com

◦ Developed in Javascript using Ionic 3 & Angular 4 frameworks.
 ◦ Email based authentication implemented using Firebase.
 ◦ Scratch and win game developed using HTML5 Canvas and event handling for various touch events.
 ◦ Push notifications integrated using One Signal API.
 ◦ Customer actions are recorded through a REST API built with NodeJS and the Express framework.

Merchant Portal for Fun Rewards
  • https://funrewards.com

◦ Merchant portal developed in Javascript using Firebase, NodeJS and MongoDB on the backend.
 ◦ Charts for analytics developed using Chartist.JS.
 ◦ Analytics performed using MongoDB Aggregation framework.
 ◦ Merchant specific images are stored securely using Firebase storage.
